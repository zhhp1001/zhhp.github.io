52. N-Queens II

依次枚举每一行皇后的位置（每一行只有一个皇后）

  1. 每一列只能有一个皇后 col[N]
  2. 每条斜线上只能有一个皇后 d[2 * N - 1], ud[2 * N - 1]

需要判断每一个坐标(x, y)在哪一条斜线上

x + y
x - y + n //因为ud[]索引非负，而x - y 可能为负， 所以这里要加上一个 n （当x = 0, y = n - 1， x - y + n = 1）
